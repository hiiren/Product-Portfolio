## Detailed Case Studies

### 01 — Fuds International: 0→1 Digital Platform
`Built & Shipped` `E-Commerce` `Home Services` `Shopify` `Google Analytics` `Figma`

- Diagnosed a dual failure: zero digital acquisition AND broken manual operations — solved both together as one connected product decision
- Built Shopify website across 7 service lines, 10+ categories, 9 brand partners in 2 weeks — solo, from Figma wireframes to live
- Made upfront pricing the core trust differentiator in a market where competitors showed none — drove 60% lead-to-job conversion
- Identified 38% mobile drop-off at date-picker via GA funnel; simplified to 4-field form — 2.5× submission lift in 4 weeks

[View PDF →](./01-Fuds-Digital-Platform.pdf)

### 02 — Fuds Field Service Dashboard: AI Operations Automation
`Built & Shipped` `AI Automation` `Vibe Coding` `AWS Amplify` `WhatsApp API` `Shopify Webhook`

- Identified 30-minute manual coordination bottleneck per order as the operations gap killing customer experience post-inquiry
- Built AI-powered dashboard using vibe coding (Claude + Antigravity) — zero engineering budget, solo build, production deployment
- Integrated Shopify webhook → admin panel → dual WhatsApp automation (customer gets verified technician details, technician gets job brief)
- Eliminated manual coordination entirely — contributing to ~40% repeat-customer retention rate

[View PDF →](./02-Fuds-Field-Service-Dashboard.pdf)

### 03 — HighProbTrade: AI Stock Analysis Platform
`Client Project` `Fintech` `Gemini AI Studio` `React` `Firebase` `NSE F&O`

- Identified gap in retail NSE F&O analysis: existing scanners lacked AI sentiment, multi-factor filtering, and actionable real-time alerts
- Designed and built a 4-phase smart scanner: market macro → sector heatmap → PDH/PDL filter → deep OI analytics across 188 F&O stocks
- Built real-time Nifty OI tracker with PCR, sentiment engine (5-level: Strongly Bullish → Strongly Bearish), and inline OI visualisation
- Delivered complete stack solo using vibe coding: React + Firebase + Gemini AI Studio — client project in testing phase ahead of launch

[View PDF →](./03-HighProbTrade-Stock-Platform.pdf)

### 04 — AI Marketing Compliance Platform
`Client Project` `InsurTech` `AI` `IRDAI` `PRD Delivered`

- Mapped features to KPIs: review cycle 3.8 days → 4 hours, violation detection 92% → 99.5%
